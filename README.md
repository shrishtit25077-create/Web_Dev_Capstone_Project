# Web_Dev_Capstone_Project
📘 CampusHub – Student Dashboard & Assignment System

CampusHub is a web-based student dashboard that allows users to manage assignments, track performance, and submit work online.
This project includes a frontend (HTML, CSS, JS) and a backend (Flask + Supabase).

🚀 Features
📊 Dashboard UI with stats (assignments, tests, scores)
📝 Assignment listing page
📤 Submit assignment with file upload
📈 Score trend & performance section
🌙 Light/Dark mode UI
🔗 Backend integration using Flask
🗄️ Database powered by Supabase
🛠️ Tech Stack
Frontend
HTML
CSS
JavaScript 
Backend
Python (Flask)
Supabase (Database + Storage)
